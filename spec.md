# Cursor Status Pet

## 项目目标

在 Windows 桌面上显示 Cursor Agent 的工作状态，即使"显示桌面"也不会被隐藏。

## 功能清单

### MUST - 核心功能

**FEAT-001: Agent 状态检测**
- 检测 Cursor 中 Agent 是否正在工作
- 区分状态：空闲、工作中
- Scenario: 用户启动 Cursor 并开始 Agent 对话，状态实时更新

**FEAT-002: 桌面状态显示 - 方案A（悬浮窗）**
- 简洁悬浮窗始终置顶显示
- 窗口可拖动定位
- 显示当前状态图标/颜色
- "显示桌面"时不被隐藏

**FEAT-003: 桌面状态显示 - 方案B（桌面宠物）**
- 小角色在桌面活动
- 不同状态对应不同动画（待机、工作中、完成）
- 可拖动位置
- "显示桌面"时不被隐藏

### SHOULD - 重要功能

**FEAT-004: 系统托盘**
- 托盘图标显示状态
- 右键菜单：切换方案A/B、设置、退出

**FEAT-005: 开机自启动**
- 可选开机自动启动

### COULD - 可选功能

- 自定义宠物外观
- 状态变化音效
- 多 Cursor 实例支持

### WON'T - 不做

- macOS/Linux 支持
- Cursor 内部插件形式
- 复杂统计面板

## 技术约束

- 平台：Windows 10/11
- 技术栈：Electron + TypeScript + React
- 窗口：使用 Windows API 实现置顶/不被最小化
- 检测：监控 Cursor 进程 + 日志文件解析

## 非目标

- 不提供 Agent 交互功能
- 不记录历史数据
